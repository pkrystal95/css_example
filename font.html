1) 상속

조상 div (.ancestor) → color: green; font-size: 18px
자식 div (.child)도 초록색 18px
손자 div (.grandchild)은 color: red; 로 덮어씀
border 속성은 상속되지 않음
→ 자식 div엔 border 없음

2) 사이즈 단위

24px — 절대 크기

2em — 부모의 2배

1.5rem — html 기준

120% — 부모 대비

em vs rem 비교

부모(.parent) font-size: 20px

2em → 20px * 2 = 40px

2rem → html 16px * 2 = 32px

3) 글꼴 속성

Georgia — 세리프 계열 (장식 있는 글꼴)

Arial — 산세리프 계열 (단순한 글꼴)

font-weight: normal

font-weight: bold

font-weight: 900 (폰트가 지원할 경우 가장 두꺼움)

font-style: italic (기울임)

line-height: 2 — 줄 간격 넓게

letter-spacing: 4px → 글자 간격 벌림
